# HR Talents Partners - Plateforme RH

## Vision
Site web professionnel pour HR Talents Partners, une agence de recrutement et conseil RH offrant des services aux entreprises et candidats. Interface publique avec espace admin pour gérer les offres d'emploi et candidatures.

## Design
- **Primary:** 0 0% 13% (charcoal - noir professionnel)
- **Background:** 45 33% 96% (warm cream - crème chaleureuse)
- **Foreground:** 0 0% 13% (dark charcoal)
- **Accent:** 43 74% 49% (gold - doré subtil pour CTAs)
- **Muted:** 45 20% 88% (beige clair)
- **Headings:** Playfair Display (élégant serif)
- **Body:** Work Sans (moderne sans-serif)
- **Style:** Élégance corporative professionnelle, espaces généreux, cadres subtils

## Features
- Page d'accueil avec hero, services, chiffres clés
- Catalogue d'offres d'emploi avec recherche/filtres
- Formulaires candidature entreprises et candidats
- Espace administrateur sécurisé
- Gestion complète des offres et candidatures
- Blog/actualités RH
